## Decisions made

**No logos, minimal styling.** Functional-first approach: system color keywords only, system font stack, no decorative elements.

**Single `node` schema for folders and notes.** Simpler tree traversal, uniform CRUD. `type` field distinguishes them.

**API keys always in localStorage, never synced.** Even when admin is logged in, Mistral and Anthropic keys stay in localStorage. The Worker never sees them.

**storage.js as the sole data access point.** `app.js`, `note.js`, `voice.js`, `chat.js` all import from storage.js only. It selects store.js (localStorage) or db.js (IndexedDB) based on sessionStorage auth token.

**Stages 10 and 12 built together.** Both live in note.js — voice append and AI edit panel are both toolbar actions in the note view. No point in separate commits.

**AI edit panel built as an in-page slide-up.** Does not navigate away from the note. Shows revised body as preview text before accept/reject. Preserves original until accepted.

**Drag reorder uses HTML5 drag events only.** No library. Position values written back to storage on drop.

**Worker uses recursive CTE for descendant deletion.** Single SQL query collects all descendants before deleting, rather than multiple round trips.
