Python code block:

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))
```

Markdown is a lightweight markup language created by John Gruber in 2004. It was designed so that it can be converted to HTML and many other formats using a tool by the same name. Markdown is widely used in blogging, instant messaging, online forums, collaborative software, documentation pages, and readme files. The initial description of Markdown contained ambiguities and raised unanswered questions, causing implementations to both intentionally and accidentally diverge from the original version. This has been partially addressed by the creation of the CommonMark specification in 2012.
