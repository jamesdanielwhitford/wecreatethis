# Session 004 — 2026-05-18

## What was done

- Switched admin auth token from `sessionStorage` to `localStorage` so login persists across tab closes and browser restarts. Logout still clears it.
- Replaced fixed `25dvh` item gap with JS-measured padding. A `ResizeObserver` on each item's `<figure>` sets `padding-top` and `padding-bottom` to `max(32px, (viewport height - figure height) / 2)`, centering each item in its own viewport slot. A `resize` listener recalculates all items on window resize.
- Added `?mock` URL flag for local development. Visiting `/symbolicritual/?mock` loads 5 hardcoded test items (landscape, portrait, square, ultrawide, tall) from `test-assets/` without touching IndexedDB or the API. Normal URL is unaffected.
- Test assets generated by `generate-test-assets.py` — 5 flat-color PNGs covering a range of aspect ratios.
- Bumped service worker to `symbolic-ritual-v9`.
- Pushed to prod.

## Decisions made

- Mock data kept in prod build — `?mock` is inert unless explicitly added to the URL, no user-facing impact.
- `ResizeObserver` watches the `<figure>` (not the `.item`) to avoid an infinite loop from padding changes triggering re-observation.
