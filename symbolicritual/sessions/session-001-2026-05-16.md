## Decisions made

- Used OpenStreetMap over Google Maps for the location link: works worldwide, no API key, no regional restrictions.
- Used `LinkText` CSS system color keyword for link styling — keeps the no-hardcoded-hex convention and automatically handles light/dark mode.
- `exifr` loaded from CDN (not bundled) — avoids adding a build step, service worker passes CDN requests through to network.
